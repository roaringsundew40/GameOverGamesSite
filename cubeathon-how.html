How to play Cube-A-Thon

November 21, 2020

Playing Cube-A-Thon is very simple thanks to the easy interface of itch.io. Itch.io is a site which offers free game hosting for small game devs (like me). Follow the directions below to download Cube-A-Thon for your Operating System.

Cube-A-Thon is a faced paced obstacle-avoidance game in which you have to go through multiple different level to beat the game. There are currently 10 different levels with updates that come soon.

Before continuing open a browser of your choice and go to this link

For Windows run the .msi file. It will install the Game Launcher. Run the Launcher let it check for updates and then you can play. Second Option for Windows is to download the Launcher. This will not install the .msi file, instead it will just install the Cube-A-Thon Launcher which removes the task of running the installer. This option will work the best with the new itch.io app (though it is not required). For MacOS move the Cube-A-Thon.app file to the Applications Directory. Then Search for it and run it. For Linux I'm not really sure how Linux apps work so just do what you will usually do on a Linux Computer.
You can play the game in the web, too.
